** Requirements

- curl 7.61.1

** Credentials

Full node URL.

#+BEGIN_SRC sh
URL="http://127.0.0.1:8090"
#+END_SRC

** Query the network
*** List nodes

#+BEGIN_SRC sh
curl $URL/wallet/listnodes | python -m json.tool
#+END_SRC

*** Get node info

#+BEGIN_SRC sh
curl $URL/wallet/getnetworkinfo
#+END_SRC

*** Get chain parameters

#+BEGIN_SRC sh
curl $URL/wallet/getchainparameters | python -m json.tool
#+END_SRC
